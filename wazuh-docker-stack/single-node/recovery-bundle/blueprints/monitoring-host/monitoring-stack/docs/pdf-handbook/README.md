# PDF Handbook

This folder contains the offline PDF export set for the project.

Expected files:

- `project-overview.pdf`
- `installation-guide.pdf`
- `troubleshooting-guide.pdf`
- `tools-user-guide.pdf`
- `access-and-credentials.pdf`
- `monitoring-and-threat-identification-guide.pdf`

These PDFs are generated from the markdown guides in `docs/operator-handbook/`.

You can read them in either of these places:

- local folder: `docs/pdf-handbook/`
- internal browser page: `http://192.168.1.3:9088`
