# Wazuh Docker Stack

A single‑node deployment of the Wazuh security platform using Docker‑Compose.

## Directory layout
```
📁 scripts/                # Helper scripts (upgrade_all.sh, rearrange.sh)
📁 single-node/            # Docker‑Compose stack and configuration
│   ├─ docker-compose.yml
│   └─ config/
│       ├─ wazuh_manager.conf
│       ├─ wazuh_dashboard/
│       │   ├─ wazuh.yml
│       │   └─ opensearch_dashboards.yml
│       └─ wazuh_indexer/
│           ├─ wazuh.indexer.yml
│           └─ ssl_certs/   # TLS certificates (git‑ignored)
📁 logs/                   # Runtime logs (upgrade_all.log)
📁 docs/                   # Documentation, diagrams, etc.
```

## Prerequisites
```bash
sudo apt-get update && sudo apt-get install -y \
    docker docker-compose sshpass curl
```

## Quick start
```bash
# Make scripts executable
chmod +x scripts/*.sh

# Run the upgrade script (it will pull and restart Pi‑hole & Mitmproxy)
bash scripts/upgrade_all.sh
```

## CI pipeline
A GitHub Actions workflow runs `shellcheck` on all shell scripts and executes a dry‑run of `upgrade_all.sh`.
